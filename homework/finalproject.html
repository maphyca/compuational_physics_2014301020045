1 引言

在重力作用且忽略摩擦力的情况下，一个质点在一点A以速率为零开始，沿某条曲线，去到一点不高于A的B，怎样的曲线能令所需的时间最短呢？这就是最速降线问题，又称最短时间问题、最速落径问题。 在部分欧洲语言中，这个问题称为Brachistochrone，即希腊语中的“最短”（brochistos）和“时间”（chronos）。通过数学求解可以得出最速降线实际上为摆线。有趣的是，等时降线的求解结果也是摆线 这意味着最速降线同时也是等时降线。

最速降线问题被看作是变分学的起源，在数学史上具有十分重要的地位。1638年，伽利略在《论两种新科学》中以为此线是圆弧。约翰·伯努利参考之前分析过的等时降落轨迹， 证明了此线是摆线，并在1696年6月的《博学通报》发表。艾萨克·牛顿、雅各布·伯努利、莱布尼兹和洛必达都得出同一结论，即正确的答案应该是摆线的一段。事实上，约翰·伯努利当时找到的证明方法是错误的。 而正确的证法是由他的哥哥雅各布发现的，在他发现以后，伯努利则将其据为己有。

在本次作业中，我们使用Scipy的优化库optimize来求解如下问题:

一个质点在在一点A以速率为零开始，去到一个与A高度相同，距离为8的点B，使得时间最短。

2 具体程序

#! usr/bin/python
#coding=utf-8   
import matplotlib.pyplot as pl
import numpy as np
import scipy.optimize as opt
import scipy.integrate as inte

N = 100.0
target = 8.0
x = np.linspace(0,target,N)
tmp = np.linspace(0, -1, N//2)
y0 = np.r_[tmp, tmp[::-1]]
g=9.8

def total_time(y):
    s = np.hypot(np.diff(x), np.diff(y))
    v = np.sqrt(2 * g *np.abs(y))
    avg_v = np.maximum((v[1:]+v[:-1])*0.5, 1e-10)
    t = s / avg_v
    return t.sum()

def fix_two_ends(y):
    return y[[0, -1]]

def brachistochrone_curve(D, N=1000):
    eps = 1e-8
    def f(y, x):
	y = min(max(eps,y), D)
	flag = -1 if x>=D * np.pi / 2 else 1
	return flag * ((D - y) / y) **0.5
    x0 = np.linspace(0, D * np.pi, N)
    y = inte.odeint(f, 0, x0)
    return x0, y.ravel()

y_opt = opt.fmin_slsqp(total_time, y0, eqcons=[fix_two_ends])
pl.plot(x, y0, "k--", label=u"initial")
pl.plot(x, y_opt,label=u"optimization")
x2, y2 = brachistochrone_curve(target / np.pi)
pl.plot(x2, -y2 , label=u" brachistochrone_curve")
pl.legend(loc="best")
pl.show()
